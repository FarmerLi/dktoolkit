多种切换效果幻灯片

示例代码

new MagicSlide({cid:'magic_slide', width: 650, height: 180, nextBar: 'next', preBar: 'pre'});

使用说明

新建幻灯片对象，通过参数配置效果。基本的html结构为： 
<div id="cid">
	<ul>
		<li><a href="#"><img src="xxx.jpg" /></a></li>
		<li><a href="#"><img src="xxx.jpg" /></a></li>
	</ul>
</div>

MagicSlide(options)

MagicSlide为幻灯片构造函数。接受一个配置参数。 默认随机获取切换效果，默认有6种切换效果，可以自行继承AbstractEffect类，实现新效果。
  1. options - (mix) 配置参数
    1. cid - (string) 幻灯片容器id
    2. width - (int) 宽度
    3. height - (int) 高度
    4. itemWidth - (int) 碎片宽度
    5. itemHeight - (int) 碎片高度
    6. nextBar - (string, optional) 下一个的按钮ID
    7. preBar - (string, optional) 上一个的按钮ID

MagicSlide(options)

注册拖动事件

参数

  1. handler - (Mix) 鼠标拖动的对象或ID
  2. operator - (Mix) 移动的对象或ID
  3. options - (json, optional) 选项
    1. onstart(event) - (function, optional) 开始拖动事件，event包含xPos位置和yPos位置
    2. onmove(event) - (function, optional) 移动事件，event包含xPos位置和yPos位置
    3. onend(event) - (function, optional) 拖拽结束事件，event包含xPos位置和yPos位置
    4. lockX - (bool, optional) 锁定X移动
    5. lockY - (bool, optional) 锁定y移动
    6. xRange - (json, optional) 包含两个值，min和max，x移动范围
    7. yRange - (json, optional) 包含两个属性，min和max，y移动范围
