local storage本地存储必须从服务器打开页面才生效

增强LocaStorage本地存储演示（支持IE6+）

示例代码

window.storage.save('username', dk.$('username').value);
window.storage.retrieve('username');
window.storage.remove('username');

使用说明

在window对象上扩展了一个storage对象。直接调用window.storage对的相关方法即可。

window.storage.retrieve( name )

按给定的名称返回获取的内容。

window.storage.save( name, value)

按照给定名称保存内容，如果已经存在则覆盖。只能存储字符串类型。

window.storage.remove( name )

按照给定名称移除内容。

window.storage.clear()

清空本地存储内容。
