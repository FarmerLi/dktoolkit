颜色选择控件

选择颜色

示例代码

dk.colorpicker('color1', {});
dk.colorpicker('color2', {simple: true});

使用说明

通过全局对象dk.colorpicker设置需要调用控件的input对象上，并传入options参数。

dk.datepicker(id[, options])

参数

  1. id - (string) 目标input的id
  2. options - (json, optional) 配置参数
    1. showNoColor - (bool, optional) 是否显示无颜色按钮
    2. onshow - (function, optional) 显示的时候调用的事件
    3. onclose - (function, optional) 关闭时调用的事件
    4. onconfirm - (function, optional) 确定时调用的事件
    5. oncancel - (function, optional) 取消时调用的事件
    6. onnocolor - (function, optional) 无颜色按钮选中时触发的事件
    7. oncolorselect - (function, optional) 颜色选中时触发的事件
